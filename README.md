# Claw Proxy
> 🛡️ Unified LLM API gateway with routing, caching, and rate limiting.

[![CI](https://github.com/mxn2020/claw-proxy/actions/workflows/ci.yml/badge.svg)](https://github.com/mxn2020/claw-proxy/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## ✨ Features
- **📡 Live Traffic** — Real-time request viewer with status, latency, tokens
- **🛣️ Routes** — Configure paths with provider fallbacks, caching, rate limits
- **🔑 API Keys** — Generate and manage proxy API keys
- **📊 Analytics** — Usage breakdown by provider
- **🌗 Dark & Light Mode** — Cyberpunk neon theme

## 🚀 Quick Start
```bash
git clone https://github.com/mxn2020/claw-proxy.git
cd claw-proxy && npm install && npm run dev
```
